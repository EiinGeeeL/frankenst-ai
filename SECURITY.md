# Security Policy

## Supported versions

Security fixes are currently applied on a best-effort basis to the latest state
of the default branch.

## Reporting a vulnerability

Do not open a public issue for undisclosed vulnerabilities.

Report security concerns privately to the maintainer email declared in
`pyproject.toml`, including:

- affected component
- impact
- reproduction steps
- any proposed mitigation

An initial acknowledgment should be expected within a reasonable best-effort
window. Once the issue is understood, remediation and disclosure timing can be
coordinated.
